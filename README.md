## 概要
 - 学習用に制作した相性診断サイトです。
 - 参考：
   - https://qiita.com/automation2025/items/06f9a781a1607b23b3a8

## フロントエンドの起動
 - cd frontend
 - npm install
 - npm start

## バックエンドの起動
 - .envファイルにAPIキー設定
 - requirements.txtをインストール
 - cd backend
 - python app.py
